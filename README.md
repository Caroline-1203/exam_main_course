# Итоговая работа по курсу основы программирования


## Задача

```
Написать программу, которая из имеющегося массива строк формирует новый массив из строк, длина которых меньше, либо равна 3 символам. Первоначальный массив можно ввести с клавиатуры, либо задать на старте выполнения алгоритма. При решении не рекомендуется пользоваться коллекциями, лучше обойтись исключительно массивами.

```

## Описание решения задачи

```
1. В начале определяется массив строк "startingArray".
2. Вызывается метод "GenerateResultArray" с передачей "startingArray".
3. В методе "GenerateResultArray" определяется количество элементов, которые удовлетворяют условию длины строки меньше или равной 3 символам, с помощью метода "CountStringSymbols".
4. Создается новый массив "resultArray" с размером, равным количеству элементов, удовлетворяющих условию.
5. Происходит итерация по "startingArray", и если длина текущей строки меньше или равна 3 символам, она добавляется в "resultArray".
6. Метод возвращает "resultArray".
7. Вызывается метод "PrintResultArray" для вывода элементов "resultArray".
8. Метод "PrintResultArray" выводит элементы массива через точку с запятой и заключает результат в квадратные скобки.

Таким образом, код фильтрует строки из "startingArray" по длине и выводит строки, длина которых меньше или равна 3 символам.
```
